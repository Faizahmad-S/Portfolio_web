Data Jobs Dashboard 2.0
  • Projects
  • Project Details

Purpose

Build an improved Power BI dashboard that helps users understand the data job market through salaries, job demand, and skill requirements.

This version was designed to provide faster, clearer, and more actionable career insights than the original dashboard.

Tools

Power BI Desktop | Power Query | DAX | Star Schema | Interactive Dashboard Design

Summary

Created a single-page Power BI dashboard using 2024 data job postings to analyze job demand, salary trends, and skill requirements in one interactive view.

Compared with Version 1.0, this update improves layout, KPI depth, skill analysis, and filtering flexibility while applying stronger data modeling and dashboard design practices.

Process

  • Imported and profiled a real-world 2024 data jobs dataset containing job roles, salaries, locations, and required skills.
  • Cleaned and transformed the data in Power Query to improve quality, consistency, and reporting readiness.
  • Built a star schema data model with optimized fact and dimension relationships for better dashboard performance.
  • Created DAX measures for KPIs such as job count, median salary, hourly rate, and average skills per job.
  • Designed a single-page interactive dashboard with slicers for Job Title and Country to support quick and personalized analysis.

Key Insights

  • The dataset includes 479K+ data job postings, showing a strong and active hiring market.
  • Jobs require an average of around 5 skills, highlighting the importance of multi-skill readiness.
  • The median yearly salary is £113.25K, with a median hourly rate of £47.62.
  • Senior Data Scientist and Machine Learning Engineer appear among the highest-paying roles.
  • Python and SQL are the most in-demand skills, while AWS and Azure strengthen market relevance.

Outcome

Built an upgraded Power BI dashboard that turns raw job-posting data into a focused career intelligence tool, helping users evaluate job demand, salary potential, and skill priorities through a cleaner and more advanced analytical experience.
