Purpose

Build an interactive Power BI dashboard that brings scattered job market information into one place for easier comparison and analysis.

The project helps users explore salaries, job titles, locations, and hiring patterns across the 2024 data job market.

Tools

Power BI | Power Query | Quick Measures | Drill-Through | Bookmarks | Data Storytelling

Summary

Created a two-page Power BI dashboard using 2024 data science job postings to analyze the data job market in a clear and interactive way.

The report combines market overview metrics with drill-through job details, helping users compare salaries, hiring platforms, remote opportunities, and geographic distribution.

Process

  • Cleaned and prepared the dataset in Power Query by handling blanks, correcting data types, and creating useful derived fields.
  • Built summary visuals such as cards, bar charts, line charts, area charts, tables, and maps to present key market patterns.
  • Designed a Market Overview page to highlight total jobs, median salaries, and top job titles in one clear view.
  • Created a Drill-Through Details page to analyze selected job titles through salary range, remote work availability, top hiring platforms, and job locations.
  • Added slicers, buttons, bookmarks, and drill-through navigation to improve usability and dashboard interactivity.

Key Insights

  • The dashboard makes it easier to compare job titles, salaries, and locations that are usually scattered across multiple sources.
  • The Market Overview page provides a fast high-level view of the data job market through headline metrics and top-role comparisons.
  • The Drill-Through page reveals deeper insights for selected roles, including salary spread, remote job share, top job platforms, and geographic concentration.
  • The project shows how interactivity in Power BI can turn raw job-posting data into a more useful decision-support tool for career exploration.

Outcome

Built a beginner-friendly but functional Power BI dashboard that transforms raw job posting data into an interactive market analysis tool for exploring data careers, salary expectations, and hiring patterns.
